Also consider: what happens when users make mistakes, use unexpected phrasing, or hit rate limits.

**Your task:** Review the feature request below. Identify the top UX concerns. Be specific — name actual user scenarios and failure modes. Consider the full journey: how does a user discover this feature, invoke it, correct mistakes, and understand the outcome?
